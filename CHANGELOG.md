# Changelog

## [0.4.0] - 2026-04-21
### Added
- 多 Agent 架构（Orchestrator + Geo/Social/Memory）
- 工具策略沙箱与高风险审批流（HITL）
- request_id 执行链路回放
- 会话压缩并导出 History Summary Markdown
- 智能助手附件上传（PDF/Excel/Word/TXT/MD/CSV）
- Token 上下文预算液位提示

### Changed
- 智能助手界面交互优化（会话、压缩、审批、调试）
- 冲突记忆处理改为审批后生效
- 上下文压缩由消息条数阈值升级为 token 阈值

### Fixed
- 多轮对话中的消息展示顺序问题
- 本轮工具调用调试面板恢复
- 文档与配置模板统一为 UTF-8 中文版
