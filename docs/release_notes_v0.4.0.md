# Release Notes v0.4.0

发布日期：2026-04-21

## 版本主题
多 Agent 架构稳定化 + 开源发布准备。

## 重点更新
1. 升级为 `Orchestrator + Geo/Social/Memory` 架构。
2. 增加工具策略沙箱与审批流。
3. 增加 request_id 执行链路回放。
4. 增加会话压缩与 History Summary Markdown 导出。
5. 增加附件上传并挂载至会话上下文。
6. 完成开源工程化文档与脚本补齐。

## 已知限制
1. 默认是本地 SQLite，不包含多用户登录体系。
2. 目前为单机部署，未接入线上任务队列和集中监控。
3. 测试以回归脚本为主，尚未引入完整 CI。

## 下一步
1. 多用户认证与权限隔离。
2. 向量记忆检索。
3. CI 自动化与容器化部署。
