# 测试用例（开源验收）

## A. 启动与环境
1. 执行 `pip install -r requirements.txt`
2. 执行 `Copy-Item .env.example .env`
3. 配置任一模型通道 API Key
4. 执行 `streamlit run streamlit_app.py`
5. 预期：主页正常打开，无白屏

## B. 路由与多 Agent
1. 在智能助手输入：`总结一下我的旅行记录`
2. 预期：路由到 `GeoAgent`（调试信息可见）
3. 输入：`帮我邀请Alina一起旅行`
4. 预期：路由到 `SocialAgent`
5. 输入：`记住我喜欢安静的地方`
6. 预期：路由到 `MemoryAgent`

## C. 工具调用与审批
1. 执行邀请类请求，触发高风险工具
2. 预期：出现“待审批工具调用”
3. 点击批准
4. 预期：状态变为已执行，日志可追踪
5. 再次触发并点击拒绝
6. 预期：状态变为已拒绝

## D. 记忆冲突
1. 输入：`我喜欢安静的地方`
2. 再输入：`我喜欢热闹的地方`
3. 预期：生成冲突待审
4. 审批新记忆后：
5. 预期：冲突旧记忆失活，其他非冲突记忆保留

## E. 历史压缩与摘要
1. 连续对话多轮后点击“压缩当前对话”
2. 选择“压缩并导出MD”
3. 预期：出现 History Summary 展示与下载按钮

## F. request_id 回放
1. 在调试信息中复制 request_id
2. 在“按 request_id 回放执行链路”中查询
3. 预期：可看到 run/tools/guards 完整链路

## G. 边界测试
1. 空输入发送
2. 预期：不触发执行
3. 上传不支持格式文件
4. 预期：被前端限制或提示失败
5. 在未配置 API Key 时提问
6. 预期：给出可理解的配置提示
